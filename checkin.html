Quick Check-in

How to Check In

  1. Ask your instructor for the class code
  2. Enter the code in the field above
  3. Answer the security question if your instructor provides one
  4. Click "Check In Now" to confirm your attendance

Tips for Successful Check-in

  • Make sure you're entering the correct class code
  • Check-in within the time window set by your instructor
  • Keep your student ID ready for verification
  • If you have issues, contact your instructor or support
